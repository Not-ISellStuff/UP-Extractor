# UP-Extractor
Tool that extracts accounts from a file

Supported Formats:

Username:LoginPage:Password
LoginPage:Username:Password
User:Email:Password

# How To Use

1. Make sure you have python 3.12 installed
2. Copy and paste the content of your file in ulp_list.txt
3. Open up main.py
